# digital-espresso-html-Dungo
This project uses a simple folder setup with four main HTML files: index.html, about.html, products.html, and branches.html. All pages are connected using relative links through a shared global navigation built with semantic nav and ul elements, so moving between pages stays consistent and easy to understand. The current page is shown as plain text instead of a clickable link to indicate the active page without using CSS. All images are placed in a separate images folder at the project root to keep media files organized and separate from the HTML files, making the project easier to manage and expand later on.
